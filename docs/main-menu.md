# Main Menu

Entry point: `python main_menu.py`

## Menu options (buttons)

These labels appear in the center list and can be clicked with the mouse:

- Overworld
- Battle Simulator
- Pixel Art Editor
- Move Animation Editor
- Map Editor
- Quit

## Keyboard controls

- Up / W: move selection up
- Down / S: move selection down
- Enter or Space: activate the selected option

## Mouse controls

- Left click on a menu option: activate it
